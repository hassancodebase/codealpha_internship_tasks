# Task 3 — Java Application using Gradle

## Overview

A Java application that calculates and displays student grades, built and managed using Gradle build automation.

## Technologies Used

* Java 25
* Gradle 9.2.0
* JUnit Jupiter 5.10.0

## Project Structure

```
src/
├── main/java/com/hassan/
│   ├── Main.java
│   ├── Student.java
│   └── GradeCalculator.java
└── test/java/com/hassan/
    └── StudentTest.java
```

## How to Run

### Run the application

```bash
.\gradlew run
```

### Run tests

```bash
.\gradlew test
```

### Build the project

```bash
.\gradlew build
```

### Clean and rebuild

```bash
.\gradlew clean build
```
